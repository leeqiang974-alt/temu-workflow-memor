# 主图 + 详情图独立工作流

## 目标

给一个产品图或产品素材库，自动生成一套电商图：

- T 首图/主图场景图。
- 高价值扩场景图。
- 产品功能/卖点详情图。
- 使用场景拼图。
- 结构细节说明图。

该流程独立于 Excel 表格处理，不能污染 D 查询库。

## 输入

- 产品图或 PNG。
- 产品标题/规格/变体信息。
- 产品分析模型输出：固定结构、可变内容、风险点、场景边界。
- 风格参考图，可来自竞品截图，但只能学习构图/空间/光线，不能复制品牌/人物/文字。

## 分析层

优先做产品分析：

- 产品是什么。
- 哪些结构必须冻结。
- 哪些内容可以替换。
- 使用场景在哪里。
- 禁止出现的场景。
- 平台风险词和风险物。

可使用：豆包多图分析、NVIDIA/Kimi/Claude Code 作为审核或规划 agent。

## 生成层

- image2/APIMart：优先测试，成本低，适合批量。
- Seedream/即梦：fallback 或高难图。
- 严格结构产品：先生成场景，再固定 PNG 合成。

## 详情图类型

1. Lifestyle 主场景图。
2. Widely Use 多场景拼图。
3. Feature Callout 细节卖点图。
4. Size/Structure 结构说明图。
5. Before/After 或收纳效果图。
6. Material/Texture 材质细节图。
7. Variant/Quantity 规格组合图。

## 安全规则

- 不用违规词。
- 不出现品牌 logo、可读商标、危险品、药品、成人/政治内容。
- 人物只能作为安全生活氛围，不遮挡/改变产品。
- 产品一致性优先级高于高级感。

## 输出

- 每个生成图记录 provider/model/source_png/prompt/local_path/status/feedback。
- 通过图进入独立主图详情图库。
- 不通过图进入 reject lock。
- 不直接写入 Excel，除非用户明确选择某图回填。
